# SEO Keyword Tracker

Restaurant SEO keyword tracking tool with Google Search Console + Analytics integration.

## Features
- 🔍 Search any restaurant → get 30 AI-generated keywords
- 🌐 Google Autocomplete integration (real search suggestions)
- 📊 Google PageSpeed Insights (real website audit)
- 📈 Google Search Console (real impressions, clicks, rankings)
- 👥 Google Analytics (real visitors, traffic sources)
- 📅 Monthly ranking tracker with charts
- 📥 CSV export
- 🌙 Dark mode UI

## Tech Stack
- Next.js 16 + TypeScript
- Turso (libSQL) database
- Z.AI GLM-4.5-flash (AI keyword generation)
- Google APIs (Search Console, Analytics, PageSpeed, Autocomplete)
- Google OAuth 2.0
- Vercel hosting
